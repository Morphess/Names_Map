# Names-Map

Simple Java console application that demonstrates usage of the `Map` interface for storing and retrieving user data.

## Task
Store a list of users (ID → Name) and:
- display all users
- get a user by specific ID


## How Works
- `DataRepository` – stores user data in a Map
- `DataHandler` – processes data and formats output
- `UIOperator` – prints results to console
- `Main` – запускає програму

## Example Output

ALL NAMES:

387, Lucy

231, Alice

394, Bob

172, Tom

NAME: id 172, Tom

## Run
Compile and run `Main.java`
